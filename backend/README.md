### Trivia App Backend

This repository is like the core of the app. once it's down, it make the app can't display anything to the user :(.

--- 

### Install

```
npm install
```

### Define the env
```
export BACKEND_PORT=""
```

### Build With docker

```
docker build -t trivia-backend:1.0.0 .
```

or 
### Run with your environment 
```
npm start
```
