### Trivia App Frontend

This repository is main display of the app. 
it communicate with back-end app and serve the message <3

--- 

### Install

```
npm install
```

### Define the env
```
export BACKEND_URI=""
export FRONTEND_PORT=""
```

### Build With docker

```
docker build -t trivia-frontend:1.0.0 .
```

or 
### Run with your environment 
```
npm start
```
